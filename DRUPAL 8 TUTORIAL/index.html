Standard Drupal 8 installation

Basic configuration

  • Click on the header to configure block
  • Set title and logo

Section "Services"

  • Content → + Add content → Basic page
  • Title → Services → Put image and text (samples here)
  • MENU SETTINGS → check "Provide a menu link"
  • URL PATH SETTINGS → URL alias → /services
  • Click "Save and publish"

    • Section "Sale of motorbikes"

  • Content → + Add content → Basic page
  • Title → Sale of motorbikes → Put image and text
  • MENU SETTINGS → check "Provide a menu link"
    Parent item → Services
  • URL PATH SETTINGS → URL alias → /sale
  • Click "Save and publish"

    • Set the main menu in the side bar

  • Structure → Block layout
    Set "Main navigation" region to "Sidebar first"

    • Expand submenu

  • Structure → Menus → Main navigation → Edit menu
    Services → Edit → Check "Show as expanded"
  • Structure → Block layout
    Main navigation → Configure → MENU LEVELS → Maximum number of menu levels to display → Unlimited

    • Sections "Mechanics workshop" and "Parts store"

  • Add sections the same way as seen before.
  • Click "Save blocks"

    • Section "Location"

  • Content → + Add content → Basic page
  • Search the address in Google Maps and obtain iframe (share option)
  • Text format → Full HTML
  • Source → Paste iframe
  • MENU SETTINGS → check "Provide a menu link"
  • URL PATH SETTINGS → URL alias → /location

    • Add news to the home page

  • Content → + Add content → Article
  • Add text and image
  • PROMOTION OPTIONS → make sure "Promoted to front page" is checked

    • Contact form

    • There is by default a contact form in /contact
    • Structure → Menus → Main navigation → Edit menu → + Add link
